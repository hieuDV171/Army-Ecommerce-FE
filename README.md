# Army Ecommerce

An ecommerce for army

## Hướng dẫn cài đặt môi trường
1. Tạo file `.env` ở thư mục gốc của dự án.
2. Copy nội dung từ file `.env.example` sang `.env`.
3. Điền địa chỉ `BASE_URL` của Server API do nhóm cung cấp.
4. Chạy lệnh `flutter pub get` để cài đặt thư viện.
5. Tạo firebase cho flutter theo hướng dẫn của firebase flutter cli.

## Code Style
1. Tất cả comment, thông báo ra màn hình đều phải là Tiếng Việt
2. Viết đoạn code nào thì comment đoạn code đó đang làm gì, nếu có logic phức tạp thì phải giải thích rõ ràng.
3. Tuân thủ coding convention
